Bonjour Mr.Roches

Prérequis : Installer NodeJS sur la machine
Remarque : Je vais placer dans le zip tout les modules nécéssaire à l'utilisation du projet, donc pas besoin d'installer quoi que ce soit
Modules utilisés: 
    Front End :
        jQuery
    Back End  :
        Express - Pour le routage
        moment - Pour facilité l'utilisation des heures
        
Instructions :
    1) Dézipper dans un dossier la totalité du contenu
    2) Lancer un invite de commande palcé à l'interieur du dossier
    3) Lancer la commande "npm start", le serveur sera alors ouvert sur le port 3000 de votre machine comme l'indique l'invite de commandes
    4) Ouvrir le index.html présent dans le dossier public/ avec votre navigateur favori(Firefox ou Chrome(non testé sur IE))

Les points manquants :
    -Les controles sont imparfaits, on peut soumettre des commandes illogiques
    -Impossible de modifier une commande existante, malgré le fait de pouvoir les visualiser.
